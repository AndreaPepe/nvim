# nvim configuration
My personal Neovim configuration.

## Plugins
All plugins are managed using the **Lazy** Plugin Manager.

### UI
- alpha-nvim: greeter
- catppuccin: colorscheme
- lualine: status line (dracula colorscheme)
- neo-tree: file explorer
- telescope: fuzzy finder
- treesitter: parsing and syntax highlighting
- which-key: keybindings popup helper
